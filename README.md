# Car Analysis Dashboard

A web application for exploring and visualizing US vehicle advertisement data.

## Description
This tool allows users to interactively explore car listing data through histograms and scatter plots. Users can filter data using checkboxes to customize the visualizations.

## Libraries Used
- **Pandas** - data manipulation
- **Streamlit** - web application framework
- **Plotly Express** - interactive charts

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/khan832130/car-analysis-dashboard.git
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
